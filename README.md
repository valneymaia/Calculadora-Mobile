# Calculadora Mobile 📱

Aplicativo mobile desenvolvido com **React Native** e **Expo**, que reúne quatro ferramentas utilitárias em uma única plataforma.

## 🚀 Funcionalidades

- **Calculadora** — Calculadora aritmética básica com suporte a adição, subtração, multiplicação e divisão.
- **IMC (Índice de Massa Corporal)** — Calcula o IMC com base no peso e altura informados, exibindo a classificação (Magreza, Normal, Sobrepeso, Obesidade ou Obesidade Grave).
- **Conversor de Temperatura** — Converte temperaturas entre Celsius, Fahrenheit e Kelvin.
- **Conversor de Moedas** — Converte valores entre USD, EUR, BTC, ETH e BRL em tempo real, utilizando a API da AwesomeAPI.

## 🛠️ Tecnologias Utilizadas

- [React Native](https://reactnative.dev/)
- [Expo](https://expo.dev/)
- [React Navigation](https://reactnavigation.org/)
- [React Native Paper](https://callstack.github.io/react-native-paper/)
- [@react-native-picker/picker](https://github.com/react-native-picker/picker)
- [AwesomeAPI](https://docs.awesomeapi.com.br/) — para cotações de moedas em tempo real

## 📋 Pré-requisitos

- [Node.js](https://nodejs.org/) instalado
- [Expo CLI](https://docs.expo.dev/get-started/installation/) instalado
- Aplicativo **Expo Go** no celular (Android ou iOS) ou um emulador configurado

## ⚙️ Instalação

1. Clone o repositório:
   ```bash
   git clone https://github.com/valneymaia/Calculadora-Mobile.git
   cd Calculadora-Mobile
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

## ▶️ Como executar

Abra um terminal na pasta do projeto e execute:

```bash
npm start
```

Isso iniciará o servidor do Expo. Escaneie o QR code com o aplicativo **Expo Go** para rodar no seu dispositivo, ou pressione `a` para abrir no emulador Android ou `i` para iOS.
